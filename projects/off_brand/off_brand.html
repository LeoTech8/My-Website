        Selecting a Random Element from Array in JavaScript

    </h2>

    <p>

        In this example, we have used <strong>Math.random()</strong>

        and <strong>Math.floor()</strong> method to select a random 

        element from array in JavaScript.

    </p>

    <button onclick="randomElement()">Select</button>

    <br><br>

    <div id="array"></div>

    <div id="result"></div>

    <script>

        const arr = [1, 2, 3, 4, 5];

        document.getElementById("array").innerHTML = "Array: " + arr;

        function randomElement() {

            const rndmEle = arr[Math.floor(Math.random() * arr.length)];

            document.getElementById("result")

                .innerHTML = "Random element from array: " + rndmEle;

        }

    </script>
